# Modern Git, Unit Testing (PHP/PHPUnit + CI3), and GitHub Copilot

This repository contains raw presentation slides in Markdown with runnable examples.

Folders:
- git/ — Git basics, CLI, and VS Code GUI
- phpunit/ — Unit testing (PHPUnit), coverage, VS Code integration
- ci3/ — CodeIgniter 3 testing (CIPHPUnitTest + manual bootstrap)
- copilot/ — GitHub Copilot usage (Web + VS Code)

How to use:
- Open each section’s README.md for slides and speaker notes.
- For phpunit/, run:
  - cd phpunit
  - composer install
  - composer test
  - composer test:coverage
